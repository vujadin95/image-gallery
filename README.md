# Image Gallery

This is simple image gallery created using Svelte compiler and Unsplash API for displaying images.

## Starting project

If you want to clone this project, you have to have Access Key from Unsplash API.

In .evn file, create variable named `PUBLIC_ACCESS_KEY` and enter your client_id for value of the variable.

```bash
# clone repository using
git clone

# install all dependencies
npm install
```

## Developing

Once you've cloned a project and installed dependencies with `npm install` (or `pnpm install` or `yarn`), start a development server:

```bash
npm run dev

# or start the server and open the app in a new browser tab
npm run dev -- --open
```

## Building

To create a production version of your app:

```bash
npm run build
```

You can preview the production build with `npm run preview`.
